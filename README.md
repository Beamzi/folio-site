# 3D Interactive Portfolio

A modern, interactive portfolio showcasing frontend development and UI/UX skills, built with React, Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Live Demo
[https://james-day.dev](https://james-day.dev)

## Tech Stack
**Frontend:** React • Next.js • TypeScript • Tailwind CSS • Framer Motion  
**3D / Animations:** Three.js  

## Features
- Fully interactive 3D portfolio experience with smooth animations  
- Custom reusable components and modular UI architecture  
- Performance-optimized for fast load times and responsive design  
- Immersive animations built with Framer Motion for engaging user experience  

## Project Highlights
- Demonstrates advanced frontend techniques including 3D scene interaction  
- Implements modern design patterns for scalability and maintainability  
- Showcases attention to detail in both design and code quality  

## Notes
Built to highlight frontend expertise and modern web development practices. Focused on creating an engaging, performant, and visually striking experience that represents both design and technical skill.
